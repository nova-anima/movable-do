# ドは、どこへでも — 移動ドの教科書とプレイグラウンド

合唱のための「移動ド」(movable do) を、無料で、ブラウザだけで身につけるための教育サイトです。

**無料・登録不要・広告なし・オープンソース。**

## なにができる?

- **教科書** — 固定ドとの違い、音の「役割」、ハモリの物理(純正律・周波数比)、短調のラ読み、ハンドサイン、4週間の練習メニュー。すべての章で音が鳴ります。
- **移調キーボード** — 12調どこでもドレミが色つきで光る鍵盤。主音ドローン、カデンツ再生、PCキー演奏対応。
- **聴音クイズ** — カデンツで調性を提示 → 1音を階名で回答。調は毎回ランダム(それが移動ドの練習)。レベル4は半音つき。
- **うたってチェック** — マイクで声の音程をリアルタイム判定(自己相関ピッチ検出)。±セント表示。音声はブラウザ内で処理され、送信されません。
- **メロディ** — パブリックドメインの名曲を任意の調で再生、階名をハイライト。「階名をかくして曲あてクイズ」も。

## 技術

- 依存ライブラリなし(Vanilla JS / ES Modules)
- 音源は Web Audio API によるリアルタイム合成(サンプル不使用)
- ピッチ検出は自己相関法 (ACF2+)
- ビルド不要 — そのまま GitHub Pages で配信

## ローカルで動かす

ES Modules を使っているため、ローカルでは簡易サーバーが必要です。

```
python -m http.server 8000
# → http://localhost:8000
```
